Instructions to use the demo application

  • Querying
    The system can handle two different types of queries to retrieve relevant handwritten documents.
    • Query by text. Use the virtual keyboard to enter a query.
      Example query word for the above collection is നിങ്ങൾ.
    • Query by example image. Upload a word image using the file upload option.
      This system can handle single word images only. Example query image for Mohanlal blog collection,
  • Results
    Top 10 relavant word images will be displayed. The results are displayed in the format shown in the image. Top left word image has the highest relevance with the query, the following image has second highest relevance and so on.
    A link is provided on top of each word image, which corresponds to the document that contains the spotted word. Click on the link to read the whole document.
