Image Restoration and Text Extraction Techniques for Images of Old Handwritten Documents



ISI Kolkata has been working on development of algorithms for restoring degraded old handwritten documents. For this a deep neural net based system has been developed which can produce a output of restored denoised image automatically if corresponding old degraded handwritten document image given as input.

A deep neural net based system has also been developed which can extract written text as output automatically if provided with the corresponding old degraded handwritten document.
